## Prerequisites

This app was built as part of project during a CMS course by Professor Emil Wcis≈Ço and Mariusz Trzaska.

## Description

The app allows viewing, adding and removing data related to gokarts which is then displayed under a
shopify  [store](https://www.turbokarting.co/pages/real-time-traffic).

It utilizes a custom .NET Web API for storing and obtaining data related to gokarts.
The Web API is hosted using a tunnelling service (ngrok), for presentation purposes.
